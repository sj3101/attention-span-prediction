# attention-span-prediction
Machine learning model that predicts user attention span based on content type, length, and user interaction patterns.

**Attention Span Prediction (ML + Flask)**

This project predicts a user's attention span based on content characteristics such as content type, length, platform, device type, time of day, and interaction behavior.
It uses a trained Machine Learning model integrated with a Flask web app to generate real-time predictions through a simple and clean UI.
